# Profit Analytics – Backend Server

## Lokal start
1. Installera beroenden: `npm install`
2. Starta servern: `npm start`
3. Servern körs på http://localhost:3001

## Deploy till Railway
1. Skapa konto på railway.app
2. New Project → Deploy from GitHub repo
3. Välj detta repo – Railway känner igen package.json automatiskt
4. Klart – Railway ger dig en publik URL
